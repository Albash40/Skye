🟦 SKYE WHITE PAPER

Version 1.0 – June 2025

🔷 1. Introduction

Skye is a decentralized Web3 platform that rewards users with SKYE tokens for completing real-world and online actions. Unlike traditional mining, Skye introduces a novel Proof of Action (PoA) system that enables users to "mine" tokens by participating in meaningful tasks such as visiting social platforms, sharing content, or interacting with dApp features. Our mission is to create a fair and engaging earning environment for all.

🔷 2. Vision & Mission

Vision: To become the leading Proof-of-Action platform where users mine value through engagement and trust.
Mission: To gamify user actions, enhance community interaction, and provide real earning opportunities through decentralized technology.

🔷 3. How Skye Works

  • Dashboard – Daily token claims, task list with rewards, and social bonus actions
  • Wallet – View SKYE balance and premium status
  • P2P Trading – Buy or sell SKYE tokens directly (admin-curated)
  • Proof of Action (PoA) – Submit secret codes to prove task completion
  • Premium – Unlock higher rewards and exclusive access
  • Play to Earn – Coming soon!

🔷 4. Proof of Action (PoA)

Skye’s PoA system replaces traditional mining with verifiable tasks.
The user is told to visit a specific link (e.g., Telegram message or social media post).
A unique code (e.g., SKYE-2050-PROOF) is displayed in that post.
The user copies the code and inputs it into the Skye app.
If the code is valid and unused, the user earns +700 SKYE instantly.
This system ensures users genuinely perform valuable actions while preventing bots or abuse.
